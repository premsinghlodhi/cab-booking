Develop a full-stack web application that enables users to book cabs, track rides in real time, and make secure online payments. The system must support multiple user roles (User, Driver, Admin) with secure authentication and role-based access control.

The application must be built using the MERN stack:

MongoDB – Database

Express.js – Backend framework

React.js – Frontend framework

Node.js – Server runtime

The system should simulate a real-world ride-booking platform with scalable architecture, secure authentication, and optimized database operations.
